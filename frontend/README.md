# Frontend
## Installation
Enter the frontend directory and run
```bash
npm install
```

## Environment Config
below is the .env file configuration
```
# backend api url
VITE_BACKEND_API_URL=https://dev.twistercambodia.com/api                        
```

## Running the app locally
To start a local server, run
```bash
npm run dev
```
