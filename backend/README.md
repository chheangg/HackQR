# Backend
## Installation

```bash
$ npm install
```

## Environment Config
below is the .env file configuration
```
SA_KEY=
PORT=
```

`SA_KEY` must file location to the Firebase key's configuration file (.json). A port is needed to run this backend server.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
